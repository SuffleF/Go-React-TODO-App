# TODO App with React and Go

I made a todo app. You can create, update and delete todos on the app.

### Setup .env file

```shell
MONGO_URI= <your_mongo_uri>
PORT= 5000
ENV= development
```

### Build the app
```shell
npm run build
```

### Backend Compile and start
```shell
go run main.go
```
